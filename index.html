Sustainable Map

The Harvard Sustainable Map was created as a resource to help people learn about sustainability at Harvard University.

Promoting a healthier, more efficient campus with sustainability.

This map was developed by Hannah Jiang, a Digital Media Design student at Harvard Extension School, as the final project for CSCE-E12.

The information and general design layout were following the Harvard Office for Sustainability website. The data for the map are from the Harvard Sustainable Walking Map, created by Eana Meng, College '19.

The map is created using Leaflet, an open-source JavaScript library for interactive maps. The tutorial for map panels creation and data import can be founded here.
